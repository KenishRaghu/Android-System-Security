# Android Systems Security Analysis Lab

**Jan 2025 – Apr 2025**

Lab project for analyzing Android system behaviors in a controlled environment, performing malware analysis, and documenting defensive improvements aligned with secure system design.

## Contents

- **[scripts/](scripts/)** — Lab code: permission analysis, exported-component checks, and package inspection (see [scripts/README.md](scripts/README.md))
- [Lab environment & security boundaries](docs/lab-environment.md) — Setup, security boundaries, and attack surfaces
- [Malware analysis & behavior inspection](docs/malware-analysis.md) — Inspection of Android app behaviors and suspicious system interactions
- [Findings & defensive improvements](docs/findings-and-improvements.md) — Documented findings and proposed mitigations

## Requirements

- Linux host (e.g., Ubuntu)
- Android emulator or device with developer options
- ADB (Android Debug Bridge), Python 3.6+
- Optional: apktool, aapt for static APK analysis
